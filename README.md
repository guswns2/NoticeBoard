## 🖥 프로젝트 소개
- 주제 : 게시판 
- 팀명 : 개인
- 인원 : 1명
- 개발 기간 : 2022.05 ~ 2023.05 (약 1주)


## ⚙ 개발 환경
- Web : jsp
- Server : Java (v1.8), Tomcat (v9.0), Spring (v4.3.2)
- DataBase : PostgreSQL
- Code Editor : STS4, DBeaver


## 🔍 주요 기능
- Login
  - ID, PW 값 DB 검증

- Sign Up
  - ID 중복체크 후 DB 저장

- Noticeboard
  - 게시글 작성, 조회, 수정, 삭제
  - 댓글 작성, 조회, 수정, 삭제
